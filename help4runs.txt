Usage: runits.sh [-h] [-b] [-s] [-j]

This script runs CapabilitiesList test
againts the system-installed jss instead
of the one just built

Capabilities run tool options:

    -h               display this help and exit
    -b               path to where tests-jss were built (such as $HOME/buildjss/jss)
    -s               path to system's slf4j jar files
    -j               path to system's jss jar file

howToList.txt shows how to list files from an installed
package on various Linux distributions.
